Landing Page – Camisetas Esportivas
Landing page responsiva para divulgação de camisetas esportivas, com foco em camisas de times de futebol, NBA e NFL. O objetivo é criar uma presença digital simples e profissional para um negócio que hoje vende principalmente por Instagram e WhatsApp.

📌 Sobre o projeto
Este projeto consiste em uma página única (landing page) que funciona como vitrine digital das camisetas, organizada em seções claras e com foco em levar o visitante até o contato via WhatsApp. Não há carrinho nem checkout: o fluxo de conversão é direto para conversa com o vendedor.

Principais objetivos:

Organizar o catálogo de camisetas de forma simples e visual.

Transmitir mais confiança para o cliente final.

Facilitar o contato via WhatsApp/Instagram para fechamento da venda.

🧩 Seções da página
A landing page é dividida em seções:

Header
Título, subtítulo e navegação por âncora para as seções da página.

Categorias
Agrupamento das camisetas por tipo (times de futebol, NBA, NFL, retrô etc.), com cards e chamadas para ver mais modelos.

Lançamentos
Destaque para peças recém-chegadas, com foco em estoque limitado e urgência.

Cultura Esportiva (NBA/NFL)
Seção voltada ao lifestyle dos esportes americanos, conectando a marca com a cultura da NBA e NFL.

O que nos diferencia
Lista de diferenciais da loja (qualidade, seleção de modelos, atendimento, confiança, variedade).

Contato / WhatsApp
Botão de WhatsApp em destaque e links para redes sociais, facilitando o contato direto para compra.

💻 Tecnologias utilizadas
HTML5 – Estrutura semântica da página.

CSS3 (Custom Properties) – Estilização, uso de variáveis CSS para cores, tipografia e espaçamento.

Responsividade com Media Queries – Adaptação para mobile, tablet e desktop.

🎨 Design e layout
Layout pensado em mobile first, ajustando para tablet e desktop com breakpoints.

Uso de variáveis CSS (:root) para padronizar a paleta de cores, fontes e espaçamentos.

Foco em visual esportivo, mas limpo, com destaque para seções de produto e chamadas para ação.

🚀 Como executar o projeto
Clone ou faça download deste repositório.

Abra o arquivo index.html diretamente no navegador ou sirva via algum servidor estático simples (Live Server, http-server etc.).

Ajuste os conteúdos (textos, imagens, links de WhatsApp e redes sociais) conforme a necessidade do cliente.

📂 Estrutura de pastas (sugerida)
index.html – Arquivo principal da landing page.

css/ – Estilos da aplicação (style.css, variáveis em :root).

assets/ – Imagens e ícones usados na página.
